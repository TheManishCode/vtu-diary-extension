Privacy Policy

Diary Made Easy [VTU]

Last updated: April 27, 2026

Local-only storage No analytics User-click affiliate links

1. Overview

Diary Made Easy [VTU] is a Chrome extension designed to help VTU students export internship diary data, generate PDF/DOC reports, and upload diary entries from JSON using the user’s active VTU session.

This policy explains what data the extension uses and how it is handled.

2. Data the extension uses

The extension may access and process the following data only for its core features:

  • Profile override values entered by the user, such as name, USN, college, and internship name.
  • Runtime logs generated during export or upload operations.
  • Diary entry data retrieved from the authenticated VTU session when the user chooses to export.
  • JSON diary entry data that the user selects for bulk upload.

3. How data is stored

Profile values and runtime logs are stored locally in Microsoft Edge extension storage on the user’s device. The extension does not send this information to third-party analytics services.

The extension does not track browsing activity outside its core functionality.

Generated files such as PDF and DOC exports are downloaded to the user’s computer through the browser.

4. Data sharing

The extension does not sell personal data. It does not share user data with advertising networks or analytics providers.

The extension communicates with VTU pages and VTU-related API endpoints only to perform the user-requested export and upload features.

The extension only uses permissions necessary for interacting with VTU pages and performing user-requested actions.

5. Affiliate links

The popup may include optional affiliate links for recommended student tools. These links open only when the user clicks them.

Affiliate links are clearly labeled inside the extension UI. They do not auto-open, auto-redirect, or inject ads into VTU websites.

6. User control

  • Users can clear logs from the popup.
  • Users can change or remove saved profile values at any time.
  • Users can uninstall the extension to remove local extension data stored by the extension.

7. Contact

For privacy questions, contact the publisher through the support contact or store listing provided with the extension.
